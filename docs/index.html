Steam Deck Verified - Optimize Arc Raiders on SteamOS

Arc Raiders is Steam Deck Verified, and the Arc Raiders Config Tuner provides full support for SteamOS and Steam Deck! The tuner automatically detects your Steam/Proton config location, making it easy to optimize DLSS, FSR, and graphics settings for the perfect handheld gaming experience.

Running on SteamOS

  1. Switch to Desktop Mode
    Press the Steam button, select Power, then "Switch to Desktop"
  2. Install GUI Support (one-time setup)
    Open Konsole and run:
    sudo steamos-readonly disable
    sudo pacman -S python python-pip python-tkinter
    sudo steamos-readonly enable
  3. Download and Run
    Download the source from GitHub, extract it, and run:
    python arc_tuner.py
  4. Auto-Detection Magic
    The tuner automatically finds your config at:
    ~/.steam/steam/steamapps/compatdata/.../pfx/drive_c/users/steamuser/AppData/Local/PioneerGame/

Linux Users: The tuner also works on desktop Linux with Steam/Proton! The same automatic config detection applies to any Linux system running Arc Raiders through Steam.
