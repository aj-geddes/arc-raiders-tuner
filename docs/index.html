ARC RAIDERS CONFIG TUNER

Tune your graphics. Dominate the battlefield.

A free, open-source tool designed to help Arc Raiders players easily configure and optimize their graphics settings. Whether you're looking to maximize performance, enhance visual quality, or fine-tune features like DLSS, FSR, Ray Tracing, and Reflex, this tool provides a simple interface to manage all your configuration options with automatic backups.

Supported Settings

Upscaling Technologies

  • NVIDIA DLSS (Deep Learning Super Sampling)
  • Intel XeSS (Xe Super Sampling)
  • AMD FSR (FidelityFX Super Resolution)

Frame Generation

  • DLSS Frame Generation
  • FSR Frame Generation

Latency Reduction

  • NVIDIA Reflex (Off, On, On + Boost)
  • AMD Anti-Lag / Anti-Lag 2

Ray Tracing

  • Ray Traced Reflections
  • Ray Traced Shadows
  • Ray Traced Ambient Occlusion
  • Ray Traced Global Illumination

Display Settings

  • Resolution
  • Display Mode (Fullscreen, Windowed, Borderless)
  • VSync
  • Frame Rate Limit

Quality Settings

  • View Distance
  • Anti-Aliasing
  • Post Processing
  • Shadows Quality
  • Texture Quality
  • Effects Quality
