Windows SmartScreen Warning

When you first run the app, Windows will show a SmartScreen warning. This is normal for open-source software that isn't signed with an expensive commercial certificate. The app is completely safe — you can review the full source code on GitHub.

1

Click "More info"

When you see "Windows protected your PC", click the "More info" link.

→
2

Click "Run anyway"

After clicking "More info", the "Run anyway" button will appear. Click it to launch the app.

Why does this happen? Code signing certificates cost $200-700/year. As an open-source project, we rely on transparency instead — the complete source code is available for anyone to review and verify.
